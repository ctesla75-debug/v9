ANDROID / GITHUB PAGES INSTALL (Full Rebuild)

1) Create a NEW folder in your repo (recommended), e.g. /v7/
2) Upload these files into /v7/:
   - index.html
   - manifest.json
   - service-worker.js
   - README-ANDROID.txt
3) Keep your existing /icons/ folder in /v7/icons/
   (icons/icon-192.png and icons/icon-512.png)

4) Commit + push.
5) On Android Chrome open:
   https://<yourname>.github.io/<repo>/v7/

Tip: Always use a NEW folder name when testing, to avoid old service-worker caches.
